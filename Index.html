Comandos Básicos Git

Comando Caracteristica Ejemplo
init Iniciar proyecto en Git git init
status verificar cambios en el archivo git status
config Obtiene y configura el repositorio o las opciones globales git config --global user.name "name/ email"
add Agregar archivos al repositorio git add nombreArchivo(agrega un archivo, git add .(agrega todos los archivos)
diff Muestra los cambios entre confirmaciones y arbol de trabajo git diff
checkout cambia de rama o restaura archivos del árbol de trabajo git checkout --nombreArchivo.ext(restaura), git checkout nombreVersión(ingresa a la versión)
branch Enumera, crea o elimina ramas(versiones) git branch (muestra), git branch nombreVersión(crear version)
.gitignore Archivo donde guardo carpetas y elementos a ignorar por git test(carpeta)
commit Registra cambios en repositorio git commit -m"comentario"
